# why python is used in DevOps

- easy to learn and read
- strong automation support
- used in ci/cd pipelines
- cloud & container friendly

# python installation

## Windows
1. Download python from python.org
2. run installer
3. check "add python to PATH"
4. verify using: python --version

## Linux
- use package manager:
  sudo apt install python3
- verify using: python3 --version

## macOS
- use homebrew:
  brew install python
- verify using: python3 --version
